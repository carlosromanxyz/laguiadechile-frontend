Instrucciones de Testing

  1. Verifica que el favicon aparece en la pestaña del navegador
  2. Prueba hacer zoom en las imágenes SVG (sin pérdida de calidad)
  3. Comprueba la visibilidad en fondos claros y oscuros
  4. En iOS, añade esta página a la pantalla de inicio para probar el Apple Touch Icon
  5. Si no ves cambios, usa Ctrl+Shift+R (Windows) o Cmd+Shift+R (Mac) para forzar actualización

Archivos Generados

  • /app/icon.svg - Versión SVG estática (recomendada)
  • /app/icon.tsx - Generador PNG 32x32 (ImageResponse)
  • /app/apple-icon.tsx - Generador PNG 180x180 (iOS)

Todos los iconos se generan automáticamente durante npm run build
