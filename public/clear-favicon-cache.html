Chrome / Edge / Brave

Method 1: Hard Refresh
  • Windows/Linux: Ctrl + Shift + Delete
  • Mac: Cmd + Shift + Delete
  • Select "Cached images and files"
  • Click "Clear data"
Method 2: Direct Cache Clear
  • Open DevTools: F12 or Cmd/Ctrl + Shift + I
  • Right-click the refresh button
  • Select "Empty Cache and Hard Reload"
Method 3: Manual Favicon Delete
  • Go to: chrome://settings/siteData
  • Search for: localhost
  • Click trash icon to delete all data for localhost
  • Restart browser

Firefox

Method 1: Clear Cache
  • Windows/Linux: Ctrl + Shift + Delete
  • Mac: Cmd + Shift + Delete
  • Select "Cache"
  • Click "Clear Now"
Method 2: Delete Favicon Database
  • Close Firefox completely
  • Navigate to Firefox profile folder:
    • Windows: %APPDATA%\Mozilla\Firefox\Profiles\
    • Mac: ~/Library/Application Support/Firefox/Profiles/
    • Linux: ~/.mozilla/firefox/
  • Delete file: favicons.sqlite
  • Restart Firefox
