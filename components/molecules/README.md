# Molecules

Combinaciones simples de atoms que forman componentes funcionales básicos.

## Propósito

- Combinan 2 o más atoms
- Tienen una función específica y clara
- Pueden tener lógica simple de interacción
- Siguen siendo reutilizables

## Componentes

- **search-bar/**: Barra de búsqueda (input + button + icon)
- **category-card/**: Tarjeta de categoría
- **rating-stars/**: Sistema de calificación con estrellas
- **form-field/**: Campo de formulario con label y validación
- **social-links/**: Enlaces a redes sociales

## Convenciones

Cada componente debe tener:
```
component-name/
├── component-name.tsx
├── index.ts
```
