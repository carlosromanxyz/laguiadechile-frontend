# UI Components (shadcn/ui)

Componentes de shadcn/ui instalados automáticamente. Estos actúan como atoms base del sistema de diseño.

## Instalación

Para agregar un nuevo componente de shadcn/ui:

```bash
npx shadcn@latest add [component-name]
```

## Componentes disponibles

Los componentes se instalan aquí automáticamente y están listos para usar.

## Uso

```tsx
import { Button } from "@/components/ui/button"
import { Input } from "@/components/ui/input"
```
