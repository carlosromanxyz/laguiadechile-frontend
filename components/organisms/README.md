# Organisms

Componentes complejos que combinan molecules y atoms para formar secciones funcionales de la interfaz.

## Propósito

- Componentes con lógica de negocio
- Combinan múltiples molecules y atoms
- Representan secciones completas de la UI
- Pueden conectarse a estado global o APIs

## Componentes

- **header/**: Encabezado del sitio con navegación
- **footer/**: Pie de página
- **business-card/**: Tarjeta de empresa con toda su información
- **filter-panel/**: Panel de filtros de búsqueda
- **navigation-menu/**: Menú de navegación principal
- **contact-form/**: Formulario de contacto completo

## Convenciones

Cada componente debe tener:
```
component-name/
├── component-name.tsx
├── index.ts
```
