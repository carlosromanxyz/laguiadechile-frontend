# Atoms

Elementos básicos e indivisibles de la interfaz. Los atoms son los componentes más pequeños y no pueden ser descompuestos en partes más simples.

## Propósito

- Componentes de UI fundamentales
- Reutilizables en toda la aplicación
- Sin lógica de negocio compleja
- Generalmente wrappean componentes de shadcn/ui o crean elementos personalizados

## Componentes

- **button/**: Botones personalizados de la marca
- **input/**: Campos de entrada personalizados
- **icon/**: Iconos personalizados y wrappers
- **badge/**: Etiquetas y badges
- **avatar/**: Avatares de usuario/empresa
- **logo/**: Logo de La Guía de Chile

## Convenciones

Cada componente debe tener:
```
component-name/
├── component-name.tsx
├── index.ts
```
